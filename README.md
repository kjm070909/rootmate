# Seomyeon Route Planner

React + Vite 기반으로 만든 카카오맵 동선 최적화 웹앱입니다. 사용자가 여러 장소를 추가하면 현재 위치를 기준으로 추천 방문 순서를 계산하고, 각 구간별 카카오맵 길안내 링크를 열 수 있습니다.

## 핵심 기능

- 키워드 검색으로 장소 추가
- 지도 클릭으로 커스텀 장소 추가
- 현재 위치 기준 방문 순서 최적화
- 총 예상 이동 거리와 도보 시간 표시
- 각 구간별 카카오맵 길안내 링크 제공

## 실행 방법

1. 프로젝트 루트에 `.env` 파일을 만들고 아래 값을 넣습니다.

```bash
VITE_KAKAO_MAP_APP_KEY=YOUR_KAKAO_JAVASCRIPT_KEY
```

2. 개발 서버를 실행합니다.

```bash
npm run dev
```

## 구현 메모

- 경로 최적화는 프런트엔드에서 `nearest neighbor + 2-opt` 방식으로 계산합니다.
- 현재 버전의 추천 순서는 직선 거리 기준입니다.
- 실제 도로 길안내는 카카오맵 링크로 연결합니다.
